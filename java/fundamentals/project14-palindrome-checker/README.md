# Java Palindrome Checker

## Description

A menu-driven application that checks whether a word, phrase, or sentence is a palindrome.

The program ignores:
- Uppercase and lowercase differences
- Spaces
- Punctuation

---

## Features

- Check words
- Check sentences
- Ignores spaces
- Ignores punctuation
- Case-insensitive
- Menu-driven interface
- Continuous execution until Exit

---

## Concepts Used

- String
- StringBuilder
- Boolean
- Methods
- Loops
- Character Class
- Input Validation

---

## Example

Input

Madam

Result

"Madam" is a PALINDROME.

---

Input

A man, a plan, a canal: Panama

Result

"A man, a plan, a canal: Panama" is a PALINDROME.

---

Input

Computer

Result

"Computer" is NOT a PALINDROME.
