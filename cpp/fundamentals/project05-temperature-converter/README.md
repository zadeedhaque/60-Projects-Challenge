
# Temperature Converter

## Description

A temperature conversion application developed as part of the 60 Project Challenge.

## Features

- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius
- Displays the formula used
- Displays both original and converted temperatures
- Menu-driven interface

## Concepts Used

- Variables
- User Input
- Conditional Statements
- Switch Statements
- Functions / Methods
- Floating Point Numbers

## Sample Output

Formula:
F = (C × 9/5) + 32

Result:
25.00 °C = 77.00 °F
