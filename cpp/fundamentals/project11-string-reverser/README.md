# C++ String Reverser

## Description

A menu-driven application that reverses any string entered by the user.

## Features

- Reverse words and sentences
- Supports spaces
- Menu-driven interface
- Input validation
- Continuous execution until Exit

## Concepts Used

- String
- Functions
- Loops
- String Indexing
- getline()
- Input Validation

## Sample Output

Original String

Hello World

Reversed String

dlroW olleH
