# C++ Multiplication Table Generator

## Description

A menu-driven multiplication table generator developed as part of the 60 Project Challenge.

## Features

- Generate multiplication tables from 1 to 10
- Input validation
- Menu-driven interface
- Continuous execution until Exit

## Concepts Used

- Variables
- Loops
- User Input
- Conditional Statements
- Input Validation
