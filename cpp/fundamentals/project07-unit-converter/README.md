# C++ Unit Converter

## Description

A menu-driven unit conversion application developed in C++ as part of the 60 Project Challenge.

The program allows users to convert between common distance and weight units while displaying the formula used for each conversion.

---

## Features

* Kilometers to Miles
* Miles to Kilometers
* Kilograms to Pounds
* Pounds to Kilograms
* Formula Display
* Original and Converted Value Display
* Input Validation
* Continuous Execution Until Exit

---

## Concepts Used

* Variables
* User Input and Output
* Loops
* Conditional Statements
* Switch Statements
* Floating Point Arithmetic
* Input Validation

---

## Conversion Formulas

### Kilometers to Miles

Miles = KM × 0.621371

### Miles to Kilometers

KM = Miles × 1.60934

### Kilograms to Pounds

Pounds = KG × 2.20462

### Pounds to Kilograms

KG = Pounds × 0.453592
